# 🛒 Styled To-Do List Website

A simple HTML and CSS shopping to-do list website that allows users to manage shopping categories and access related shopping websites directly.

## 📌 Project Overview

This project displays a shopping list with checkboxes and clickable links for each shopping category.

Users can:
- View shopping categories
- Mark items using checkboxes
- Open related shopping websites in a new tab
- Practice HTML structure and CSS styling


## ✨ Features

- Semantic HTML structure
- Styled shopping list layout
- Checkboxes for task tracking
- Nested lists for sub-items
- External links using `target="_blank"`
- Separate CSS file for styling


## 🛠 Technologies Used

- HTML5
- CSS3

## 🚀 How to Run

1. Download or clone the project.
2. Open the project folder in VS Code.
3. Open `index.html`.
4. Right-click → **Open with Live Server**
   *(or double-click the HTML file)*
